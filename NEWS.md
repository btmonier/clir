# clir 0.1.0

- Initial release.
- Added `ArgumentParser()`, `add_argument()`, `parse_args()`, and `print_help()` for building command-line interfaces in R.
- Supports common argument features including defaults, type conversion, choices validation, required flags, `nargs` (`?`, `+`, `*`), and `action` (`store_true`, `store_false`, `count`).
- Added a getting-started vignette.
